Key Features

Markdown Powered
Write documentation in standard Markdown with frontmatter support. Fast and familiar.
Multi-Language
Built-in i18n support with English and Chinese out of the box. Easy to extend.
Full-Text Search
Instant client-side search across all documentation. Find what you need, fast.
Dark Mode
Automatic dark mode with system preference detection. Easy on the eyes, day or night.
SSG + Hydration
Pre-rendered static HTML with React hydration for instant loads and SEO-friendly pages.
Shiki Highlighting
Beautiful dual-theme syntax highlighting for code blocks via Shiki, with dark/light mode.
KaTeX Math
Fast mathematical typesetting with KaTeX for inline and block-level formulas.
Mermaid Diagrams
Lazy-loaded Mermaid diagrams for flowcharts, sequence diagrams, and more.
Responsive Design
Collapsible sidebar with overlay on mobile. shadcn/ui powered responsive layout.
SEO Friendly
Pre-rendered static HTML ensures search engines can index every page perfectly.
Easy Deploy
Build to pure static files with Vite. Deploy to any CDN or static hosting in seconds.
Multiple Themes
Easily modify the color scheme to achieve a variety of theme styles.
